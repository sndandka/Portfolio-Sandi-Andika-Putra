  • Sistem kalkulasi denda otomatis terkomputasi berbasis hari
  • Laporan Exportable yang teroptimasi untuk cetak fisik
  • UI responsif modern berbasis Tailwind CSS

Mobile 2025

Dompetku - Personal Finance App

Aplikasi manajemen keuangan pribadi (Personal Finance) komprehensif berbasis Android. Membantu pengguna memantau pemasukan dan pengeluaran secara terstruktur, mengelola tagihan, dengan sinkronisasi data cloud real-time via Firestore.

✨ Fitur Utama
  • Cloud Sync & CRUD transaksi multi-akun dengan Firebase Firestore
  • Pengelolaan tagihan rutin (Bill Tracking) dengan Notifikasi push cerdas (AlarmManager)
  • Dashboard interaktif untuk memonitor total aset dan rangkuman keuangan
  • Laporan analitik interaktif dalam bentuk Line Chart & Pie Chart dinamis
  • Arsitektur MVVM (Model-View-ViewModel) dengan Material Design 3
JavaFirebase FirestoreAndroid SDKMVVMMPAndroidChart

✨ Fitur Utama

  • Implementasi pencahayaan interaktif (GL_LIGHTING)
  • Navigasi dan kamera interaktif 3D
  • Sistem render realtime dengan C++

Sistem Inventaris Sekolah MySQL

Aplikasi desktop berbasis Visual Basic .NET yang terintegrasi dengan database MySQL untuk mengelola data inventaris di lingkungan sekolah secara digital. Memudahkan pencatatan, pembaruan, pelacakan barang-barang fasilitas maupun aset sekolah.

Desktop App 2024

LiteKasir

LiteKasir adalah aplikasi kasir (Point of Sales) sederhana berbasis desktop yang dirancang menggunakan Java murni. Fokus pada kemudahan pemilik usaha kecil dan menengah (UMKM) dalam mengelola transaksi penjualan harian, stok barang, dan pencatatan komprehensif.

✨ Fitur Utama
  • Input dan manajemen data produk/barang
  • Manajemen stok otomatis terpotong setiap terjadi transaksi
  • Proses transaksi pos (Point of Sales) interaktif
  • Menyediakan laporan penjualan dan rekap stok
  • Antarmuka aplikasi yang sederhana (User-Friendly)
